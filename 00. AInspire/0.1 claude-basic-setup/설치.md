# 설치 안내 — Claude가 읽고 그대로 실행하세요

이 폴더는 **Claude Code 기초 설정 패키지**입니다. 사용자가 "claude.md랑 설치.md 읽고 세팅해줘"라고 하면, 아래를 순서대로 직접 수행하세요.

## 0. OS별 홈 경로 확인
- Windows: `C:\Users\<사용자명>\.claude\`
- macOS / Linux: `~/.claude/`

## 1. 전역 규칙 적용 (CLAUDE.md)
- 이 폴더의 `CLAUDE.md`를 사용자 홈의 `~/.claude/CLAUDE.md`에 적용한다.
- 기존 파일이 있으면 덮어쓰지 말고 `CLAUDE.md.bak`으로 백업한 뒤 내용을 병합한다. 충돌하면 사용자에게 확인한다.

## 2. 전역 권한 설정 (settings.json)
- `settings.json.template`을 `~/.claude/settings.json`으로 적용한다.
- 기존 `settings.json`이 있으면 `permissions.allow` 항목만 병합하고, API key 등 기존 값은 절대 지우지 않는다.

## 3. 스킬 설치
- `skills/agent-team`, `skills/background-run` 폴더를 `~/.claude/skills/`로 복사한다.
- 각 `SKILL.md` 맨 위에 YAML frontmatter(`name`, `description`)가 없으면, 파일 내용을 요약해 추가한다. (frontmatter가 없으면 Claude Code가 스킬로 인식하지 못함)

## 4. 확인 및 보고
- 설치가 끝나면 스킬이 정상 등록됐는지 확인하고, 결과를 한국어로 짧게 보고한다.
