# 설치 안내 — AI 에이전트가 읽고 그대로 실행하세요

이 폴더는 **Premiere Pro MCP 자동 제어 패키지**입니다. 사용자가 "설치.md 읽고 세팅해줘"라고 하면, 사용자 대신 설치를 진행하세요. Claude Code, Codex, Cursor 등 어떤 에이전트가 읽어도 절차는 동일합니다.

## 1. 사전 확인
- Node.js 18+ 설치 여부를 확인한다. 없으면 사용자에게 nodejs.org에서 LTS 설치를 안내하고 대기한다.
- 사용자 OS(Windows / macOS)를 확인한다.

## 2. 설치 실행
이 폴더 경로에서 OS에 맞는 명령을 실행한다. 이 스크립트가 의존성 설치(`npm install`) · 빌드(`npm run build`) · CEP 확장 복사 · **Claude Desktop** MCP 등록까지 자동 수행한다.
- Windows: `npm run setup:win`
- macOS: `npm run setup:mac`

## 3. 지금 실행 중인 클라이언트에 MCP 등록
위 스크립트는 **Claude Desktop에만** 등록한다. 에이전트인 너는 자신이 어떤 클라이언트에서 실행 중인지 판단해서, 그 클라이언트에도 등록해야 한다.

치환 규칙:
- `<폴더경로>` = 이 폴더의 절대 경로 (예: `C:\Users\me\Downloads\claude-premiere-mcp`)
- `<TEMP>` = Windows면 `%TEMP%\premiere-mcp-bridge`의 실제 값 (예: `C:\Users\me\AppData\Local\Temp\premiere-mcp-bridge`), macOS면 `/tmp/premiere-mcp-bridge`

클라이언트별 등록 명령:

- **Claude Code**:
  ```
  claude mcp add premiere-pro -s user -e PREMIERE_TEMP_DIR=<TEMP> -- node <폴더경로>/dist/index.js
  ```
- **Codex**:
  ```
  codex mcp add premiere_pro --env PREMIERE_TEMP_DIR=<TEMP> -- node <폴더경로>/dist/index.js
  ```
- **Claude Desktop**: 2단계 스크립트가 이미 등록함. 추가 작업 없음.
- **그 외 클라이언트** (Cursor, Gemini CLI 등): 해당 클라이언트의 MCP 설정 파일(JSON)에 아래 항목을 추가한다.
  ```json
  {
    "mcpServers": {
      "premiere-pro": {
        "command": "node",
        "args": ["<폴더경로>/dist/index.js"],
        "env": { "PREMIERE_TEMP_DIR": "<TEMP>" }
      }
    }
  }
  ```

등록 후 클라이언트를 **재시작**해야 MCP 도구가 보인다고 사용자에게 안내한다.

## 4. 프리미어 브릿지 (사용자 직접 단계)
설치가 끝나면 사용자에게 안내한다:
1. 프리미어 프로 실행(켜져 있었다면 재시작) → 메뉴 `창(Window) → 확장(Extensions) → MCP Bridge (CEP)`
2. 패널의 **Temp Directory**가 위 `<TEMP>` 경로와 같은지 확인 → **Save Configuration** 클릭
3. **Start Bridge** 클릭 → **Test Connection** 클릭
4. 실패하면 **Run Diagnostics** 클릭 후 생성되는 진단 파일(`<TEMP>/premiere-mcp-diagnostics-latest.json`)을 에이전트에게 보여달라고 한다.

## 5. 점검 및 보고
- 설치 후 직접 검증한다: `dist/index.js` 생성 여부, CEP extensions 폴더에 `MCPBridgeCEP` 복사 여부, 자기 클라이언트에 `premiere-pro` 등록 여부.
- 문제 발생 시 `npm run setup:doctor:win`(또는 `setup:doctor:mac`)으로 진단한다.
- 결과를 한국어로 짧게 보고한다.
