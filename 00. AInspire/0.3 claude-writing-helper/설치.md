# 설치 안내 — Claude가 읽고 그대로 실행하세요

이 폴더는 **글쓰기 & 프롬프트 검수 패키지**입니다. 사용자가 "설치.md 읽고 세팅해줘"라고 하면, 아래를 직접 수행하세요.

## 0. OS별 홈 경로 확인
- Windows: `C:\Users\<사용자명>\.claude\skills\`
- macOS / Linux: `~/.claude/skills/`

## 1. 스킬 설치
- `skills/humanizer`, `skills/translator` 폴더를 사용자 홈의 `~/.claude/skills/`로 복사한다.
- `humanizer` 폴더 안에 `.git` 폴더가 있으면 복사에서 제외한다(불필요).
- 각 `SKILL.md`의 frontmatter(`name`, `description`)를 유지한다.

## 2. 확인 및 보고
- 설치 후 두 스킬이 정상 등록됐는지 확인하고, 결과를 한국어로 짧게 보고한다.
