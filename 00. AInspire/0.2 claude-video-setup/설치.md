# 설치 안내 — Claude가 읽고 그대로 실행하세요

이 폴더는 **AI 영상 제작 & 자동화 패키지**입니다. 사용자가 "설치.md 읽고 세팅해줘"라고 하면, 아래를 직접 수행하세요.

## 0. OS별 홈 경로 확인
- Windows: `C:\Users\<사용자명>\.claude\skills\`
- macOS / Linux: `~/.claude/skills/`

## 1. 스킬 설치
- `skills/seedance-prompt` 폴더를 사용자 홈의 `~/.claude/skills/`로 복사한다.
- `SKILL.md`의 frontmatter(`name`, `description`)를 유지한다. 누락 시 추가한다.

## 2. 확인 및 안내
- 설치 후 스킬의 실제 등록 이름을 확인해 사용자에게 알려준다.
- `seedance-prompt`는 Jimeng Seedance 2.0 영상 프롬프트 작성을 돕는 스킬임을 안내한다.
